# 🎮 GUI Hangman Game

A fun and interactive **GUI-based Hangman Game** built with **Python** using the **Tkinter** library. Test your vocabulary by guessing the hidden word before you run out of attempts!

---

## 📌 Features

- 🖥️ User-friendly graphical interface
- 🎲 Random word selection
- ❤️ Limited number of lives
- 🔤 Letter-by-letter guessing
- 🚫 Prevents repeated guesses
- 🎨 Dynamic hangman drawing after each incorrect guess
- 🏆 Win/Lose message at the end of the game
- 🔄 Restart game option

---

## 🛠️ Technologies Used

- Python 3.x
- Tkinter (GUI)
- Random module

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Hangman-Game.git
```

2. Navigate to the project folder

```bash
cd Hangman-Game
```

3. Run the game

```bash
python main.py
```

---

## 🎯 How to Play

1. Start the game.
2. A random word is selected.
3. Guess one letter at a time.
4. Correct guesses reveal the letters.
5. Incorrect guesses reduce your remaining lives and update the hangman image.
6. Guess the complete word before all lives are lost.
